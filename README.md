# Open Calendar

桌面智能日历 Web App —— 融合中国传统历法、八字命理与 AI 对话的桌面玄学伙伴。

## 功能

- **首页日历** — 阳历/农历/干支日期、今日十神、宜忌、幸运色/幸运食物
- **今日运势** — 感情/财富/事业/学业四维运势详情，AI 报告 + 语音播报
- **摇卦占卜** — 六爻摇卦、卦象解读、AI 解签、语音对话
- **特殊日提醒** — 节气、干支关系、十神流日的个性化解读
- **语音对话** — 豆包实时语音大模型，支持多轮对话
- **投屏模式** — 角色动画投屏到副屏/角色舱
- **待办日历** — 按日期管理待办事项
- **热敏打印** — 签文/标签打印输出
