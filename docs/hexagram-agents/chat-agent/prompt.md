# 身强身弱｜极简两步打分（S）

> 只用 **月令 + 根** 两件事。  
> 不数十神、不查藏干、不开复杂表。

## Step 1：月令分（最关键）
先认五行旺季（只背这一行就够）：
- 木旺：寅卯｜火旺：巳午｜金旺：申酉｜水旺：亥子｜土旺：辰戌丑未

给分：
- 月支 = 日主同五行旺季 → `+3`
- 月支 = 生我之季 → `+1`
- 月支 = 克我之季 → `-1`
- 其他 → `0`

## Step 2：根分（只看地支"有没有同类"）
看四个地支里，有没有"日主同五行旺季的支"（同类根）：
- 有 2 个及以上 → `+2`
- 有 1 个 → `+1`
- 0 个 → `0`

## 总分
`S = 月令分 + 根分`

## 判定
- `S >= 4` → 身强
- `S <= 1` → 身弱
- `S = 2~3` → 中和（偏强/偏弱看大运再定）
