## Skill2:财运、求财、金钱相关知识
1. 财星判定流程
   财星的定义：五行中被我克制的是财星，例如日主五行属水，水克火，那么火就是财星。
   查看原局，是否存在财星。
   1.1 有财星
   判断是否透出：
   若透出，则观察财星所在柱，分析其活跃时期。
   1.2 无财星
   检查藏干中是否存在财星：
   有藏财：
   检查大运是否引发藏财。
   若大运无助，则看流年是否能引出藏财。
   1.3 完全无财：
   通过命局的其他寻找财富途径：
   食神：通过思维能力、创意和想法赚取财富。
   伤官：依靠动手能力、创新能力获取财富。

2. 财星类型分析
   正财
   特征：固定收入来源，象征稳固之财，最佳位置在日支，助婚姻与家庭财富积累。
   偏财
   特征：浮动收入来源，象征意外之财，天干现偏财称为"天财"，往往暗示机遇型财富，如投资、投机收入。

3. 财星位置解析
   年柱：
   财运来自远方，适宜去外地或远方寻求财富。
   月柱：
   主要经济来源或财运的基础。
   日柱：
   财运与情感密切相关，因伴侣或感情关系获益。
   时柱：
   晚年财运较佳，财富积累多集中于晚年时期。
